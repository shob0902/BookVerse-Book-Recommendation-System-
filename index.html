About The Project

BookVerse is an intelligent book recommendation system built with Python and Flask. It uses collaborative filtering and machine learning algorithms to provide personalized book recommendations based on user preferences and reading patterns.

Key Features

  • Personalized book recommendations
  • Top 50 trending books showcase
  • Smart search functionality
  • User ratings and feedback system
  • Modern, responsive design
